# FrenchAnki
Programme to fetch Anki definitions for French, using WordReference.com (for sentences and definitions), Wiktionary.com for pronunciations, Duckduckgo image search for images and Forvo for audio (requires API key).

**Cards produced with the following fields:**
Sentence (with word blanked out).
Picture
Definition,
Back (a single word or phrase),
Gender,
Full sentence,
Extra info (pronunciation, etc.),
Make two cards,
Test Spelling,
Test Pronunciation,
Test Gender,
Copy and Paste Area

## Getting started

Install all requirements with

```bash
pip install -r requirements.txt
```
